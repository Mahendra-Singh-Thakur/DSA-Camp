Bubble Sort

Bubble sort is a simple sorting algorithm that repeatedly steps through the list to be sorted, compares each pair of adjacent items, and swaps them if they are in the wrong order. In this variation of bubble sort, the largest elements "bubble" to the top of the list with each iteration. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted.

Steps to Perform Bubble Sort:

  1. Start from the first element in the list.
  2. Compare the current element with the next element.
  3. If the current element is greater than the next element, swap them.
  4. Move to the next element and repeat steps 2 and 3 until the end of the list.
  5. Repeat steps 1-4 until no more swaps are needed, indicating that the list is sorted.

Time Complexity:

Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. In the worst-case scenario, where the input array is in reverse order, bubble sort will require O(n^2) comparisons and swaps, where n is the number of elements in the array. This worst-case time complexity occurs because, in each pass through the array, the algorithm may need to perform a swap for every pair of adjacent elements.

In the best-case scenario, where the input array is already sorted, bubble sort will require O(n) comparisons. However, even in the best case, it will still require O(n^2) swaps because it needs to move the largest element to its correct position in each pass through the array.

The average-case time complexity of bubble sort is also O(n^2), making it inefficient for large datasets.

Space Complexity:

Bubble sort is an in-place sorting algorithm, meaning it doesn't require any extra space beyond the array being sorted. Therefore, the space complexity of bubble sort is O(1), indicating constant space usage, regardless of the size of the input array.

Bubble Sort Algorithm

                // Input: array - the array to be sorted
                Procedure bubbleSort(array)

                    DECLARE n = LENGTH(array) // Length of the array
                    DECLARE swapped // Flag to indicate if any elements were swapped

                    FOR (i = 0) TO (n - 1) DO
                        swapped = FALSE // Reset swapped flag for each pass

                        FOR (j = 0) TO (n - i - 1) DO
                            // Compare adjacent elements and swap if necessary
                            IF array[j] > array[j + 1] THEN
                                SWAP(array[j], array[j + 1]) // Swap elements
                                swapped = TRUE // Set swapped flag to true
                            END IF
                        END FOR

                        // If no elements were swapped, the array is already sorted
                        IF NOT swapped THEN
                            BREAK // Exit the outer loop
                        END IF
                    END FOR
                End Procedure
