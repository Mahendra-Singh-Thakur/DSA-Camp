Counting Sort

Counting sort is a non-comparison-based sorting algorithm that works by counting the number of occurrences of each unique element in the input array and using arithmetic to determine the position of each element in the sorted output array.

Steps to Perform Counting Sort:

  1. Find the maximum element in the input array to determine the range of values.
  2. Create a counting array to store the count of each unique element.
  3. Iterate through the input array and update the counting array.
  4. Modify the counting array to store the position of each element in the output array.
  5. Construct the sorted output array based on the counting array.

Counting Sort Algorithm

        // Input: array - the array to be sorted
        Procedure countingSort(array)
            
            DECLARE n = LENGTH(array) // Length of the array
            DECLARE maxElement = MAX(array) // Maximum element in 
                the array
            
            // Create a counting array with size maxElement + 1
            DECLARE countingArray = ARRAY of SIZE (maxElement + 1)
            
            // Initialize countingArray with zeros
            FOR (i = 0) TO maxElement DO
                countingArray[i] = 0
            END FOR
            
            // Update countingArray with the count of each element
            FOR (i = 0) TO (n - 1) DO
                countingArray[array[i]] = countingArray[array[i]] + 1
            END FOR
            
            // Modify countingArray to store the position of each element
            FOR (i = 1) TO maxElement DO
                countingArray[i] = countingArray[i] + countingArray[i - 1]
            END FOR
            
            // Construct the sorted output array
            DECLARE sortedArray = ARRAY of SIZE n
            FOR (i = n - 1) DOWN TO 0 DO
                sortedArray[countingArray[array[i]] - 1] = array[i]
                countingArray[array[i]] = countingArray[array[i]] - 1
            END FOR
            
            // Copy the sorted array to the input array
            FOR (i = 0) TO (n - 1) DO
                array[i] = sortedArray[i]
            END FOR
        End Procedure
