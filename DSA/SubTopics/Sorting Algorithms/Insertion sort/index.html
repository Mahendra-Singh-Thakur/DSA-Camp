Insertion Sort

Insertion sort is a simple sorting algorithm that builds the final sorted array one item at a time. It works by taking one element from the input data and inserting it into the correct position in the already sorted portion of the array.

Steps to Perform Insertion Sort:

  1. Start from the second element in the list.
  2. Iterate over each element starting from the second one.
  3. For each element, compare it with the elements to its left and move it to the appropriate position in the sorted segment.

Time Complexity:

Insertion sort is a simple comparison-based sorting algorithm that builds the final sorted array one element at a time. It repeatedly takes an element from the unsorted part of the array and inserts it into its correct position in the sorted part of the array.

The time complexity of insertion sort is O(n^2) in the worst-case scenario, where n is the number of elements in the input array. This worst-case time complexity occurs when the input array is in reverse sorted order, and each element must be compared and moved to its correct position in the sorted array.

However, in the best-case scenario, where the input array is already sorted, insertion sort has a time complexity of O(n). This best-case time complexity occurs because each element is already in its correct position and no element needs to be moved.

In average cases, insertion sort also has a time complexity of O(n^2), making it inefficient for large datasets.

Space Complexity:

Insertion sort is an in-place sorting algorithm, meaning it doesn't require additional space beyond the input array itself. Therefore, the space complexity of insertion sort is O(1), indicating constant space usage, regardless of the size of the input array.

Insertion Sort Algorithm

                        // Input: array - the array to be sorted
                        Procedure insertionSort(array)

                            DECLARE n = LENGTH(array) // Length of the array
                            DECLARE key, j // Variables for the key and inner loop index

                            // Iterate over the array starting from the second element
                            FOR (i = 1) TO (n - 1) DO
                                key = array[i] // Current element to be inserted
                                j = i - 1 // Index of the last sorted element

                                // Move elements of array[0..i-1], that are greater than key, to one
                                        position ahead of their current position
                                WHILE (j >= 0 AND array[j] > key) DO
                                    array[j + 1] = array[j]
                                    j = j - 1
                                END WHILE

                                array[j + 1] = key // Insert the key element into the sorted segment
                            END FOR
                        End Procedure
