Bucket Sort

Bucket sort is a sorting algorithm that works by distributing the elements of an array into a number of buckets. Each bucket is then sorted individually, either using a different sorting algorithm or recursively applying the bucket sorting algorithm itself. After all the individual buckets are sorted, the buckets are concatenated to form the final sorted array.

Steps to Perform Bucket Sort:

  1. Create an array of empty buckets.
  2. Iterate through the input array and distribute elements into buckets based on their value.
  3. Sort each bucket individually, either using another sorting algorithm or recursively applying bucket sort.
  4. Concatenate all the sorted buckets to get the sorted array.

Time Complexity:

The time complexity of bucket sort depends on the algorithm used to sort each bucket. If a stable sorting algorithm like Insertion Sort is used for sorting each bucket, and if the elements are uniformly distributed across the buckets, then the average time complexity of bucket sort is O(n+k), where n is the number of elements and k is the number of buckets. However, the worst-case time complexity can be higher depending on the distribution of elements.

Space Complexity:

The space complexity of bucket sort depends on the number of buckets used. If k buckets are used, the space complexity is O(n+k).

Bucket Sort Algorithm

Procedure bucketSort(array)
    // Description: Sorts an array using the bucket sort algorithm
    // Input: array - the array to be sorted
    DECLARE n = LENGTH(array) // Length of the array
    DECLARE numBuckets = 5 // Number of buckets

    // Create buckets
    buckets = new Array(numBuckets)
    FOR i = 0 TO numBuckets - 1 DO
    {
        buckets[i] = new Array()
    }
    END FOR

    // Place elements into buckets
    FOR i = 0 TO n - 1 DO
    {
        bucketIndex = floor(array[i] * numBuckets /
            (MAX_ELEMENT + 1))
        buckets[bucketIndex].push(array[i])
    }
    END FOR

    // Sort each bucket
    FOR i = 0 TO numBuckets - 1 DO
    {
        buckets[i].sort()
    }
    END FOR

    // Concatenate buckets
    sortedArray = new Array()
    FOR i = 0 TO numBuckets - 1 DO
    {
        sortedArray = sortedArray.concat(buckets[i])
    }
    END FOR

    // Copy the sorted array back to the input array
    FOR i = 0 TO n - 1 DO
    {
        array[i] = sortedArray[i]
    }
    END FOR
End Procedure
