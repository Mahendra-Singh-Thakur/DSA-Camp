Binary Search

Binary search is an efficient search algorithm that is used to find a specific element within a sorted array or list. It works by repeatedly dividing the search interval in half until the target element is found or the interval is empty.

Steps to Perform Binary Search:

  1. Initialize two pointers, left and right, to the first and last elements of the array, respectively.
  2. While the left pointer is less than or equal to the right pointer:
    • Calculate the middle index of the current interval as the average of the left and right pointers.
    • If the target element is equal to the middle element, return the index of the middle element.
    • If the target element is less than the middle element, update the right pointer to mid - 1.
    • If the target element is greater than the middle element, update the left pointer to mid + 1.
  3. If the target element is not found after the loop, return a sentinel value indicating that the element is not present in the array.

Time Complexity:

Binary Search has a time complexity of O(log n), where n is the number of elements in the array. This time complexity arises from the fact that the search space is halved in each iteration of the algorithm. As a result, Binary Search is highly efficient, especially for large datasets.

Space Complexity:

Binary Search has a space complexity of O(1), indicating constant space usage. This is because the algorithm does not require additional space beyond a few variables used for storing indices and temporary values. Thus, Binary Search is efficient in terms of memory usage as well.

Binary Search Algorithm

// Input: sortedArray - the array to be searched
//        target - the element to search for
Procedure binarySearch(sortedArray, target)

    DECLARE left = 0 // Leftmost index of the array
    DECLARE right = LENGTH(sortedArray) - 1 // Rightmost index of the array

    WHILE left <= right DO
        DECLARE mid = left + (right - left) / 2 // Calculate the middle index

        // If the target is found at the middle
        IF sortedArray[mid] = target THEN
            RETURN mid // Return the index of the target
        // If the target is in the left half of the array
        ELSE IF sortedArray[mid] > target THEN
            right = mid - 1 // Update the right boundary
        // If the target is in the right half of the array
        ELSE
            left = mid + 1 // Update the left boundary
        END IF
    END WHILE

    // If the target is not found in the array
    RETURN -1 // Return a sentinel value indicating the target is not found
End Procedure
